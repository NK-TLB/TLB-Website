# The Laundry Bag — Laundry Unit Photo & Video Shoot Strategy

A complete, production-ready plan for shooting **one flagship laundry unit** so the
footage powers a premium, B2B website (and social, decks, PR). Built to feed the
site's existing responsive image/video pipeline (`scripts/process-press-media.mjs`,
WebP/JPEG `srcset`, mp4/webm) and to slot directly into known website sections.

> Legend: `▢` photo frame · `🎥` video/b-roll · `●` subject · `◀` overlay/graphic ·
> `»` motion/camera move · `[ ]` on-screen text.

## 1. Goals & how shots map to the website

**Primary goal:** replace stock/parallax imagery with authentic, premium footage of a
real TLB unit that signals scale, hygiene and professionalism to **procurement buyers**
at hotels and hospitals.

Every shot must have a destination. Target placements:

| Website section | What it needs | Shot type |
|---|---|---|
| Home hero | 1 cinematic wide of the floor | 16:9 photo + 🎥 hero loop |
| Home/Commercial service cards | Washing, finishing, dry-clean, dispatch | 4:3 photos |
| "How we partner" (4 steps) | Assess → process → finish → deliver | 4 matching 4:3 photos |
| Healthcare linen | Infection-safe handling, whites | 4:3 + detail |
| Hospitality linen | Crisp white sheets, folding, banquet | 4:3 + detail |
| Certifications & equipment | Machine row, dials, ISO board | 16:9 + detail macros |
| About / leadership | Founder + team on floor | 4:3 portrait + candid |
| Counters / parallax band | 1 darkenable wide (text overlays) | 16:9, low-detail center |
| Testimonials bg | Soft, blurred floor texture | 16:9 |
| Social / reels | Vertical process clips | 🎥 9:16 |

## 2. The aspect-ratio contract (shoot for these)

The site serves responsive crops; shoot **loose** so we can crop without losing subjects
(this is exactly the bug we just fixed on the Ratan Tata photo — never fill the frame
edge-to-edge with the subject).

- **16:9** — heroes, parallax bands, video. (Primary.)
- **4:3** — service/feature cards, gallery. (Most photos.)
- **1:1** — leadership portrait, some social.
- **9:16** — Instagram/YouTube reels b-roll.

```
  Frame discipline — leave safe margins:
  ┌─────────────────────────────┐
  │   · · · · · · · · · · · ·    │  ← 12% headroom / breathing space
  │   ·   ┌───────────────┐ ·    │
  │   ·   │   ● SUBJECT    │ ·    │  ← keep subject inside inner box
  │   ·   └───────────────┘ ·    │     so 16:9 AND 4:3 crops both work
  │   · · · · · · · · · · · ·    │
  └─────────────────────────────┘
  Shoot 1 wide "master" of each scene, then push in for the 4:3.
```

## 3. Pre-production

**1–2 weeks before**
- Pick the **best-looking unit** (newest machines, most space, best light). Confirm a
  date during a **normal working shift** (we want real activity, not an empty room).
- Walkthrough/recce with the photographer: note light direction by hour, power points,
  cramped angles, ceiling height, reflective/steamy zones.
- Confirm crew, talent (staff who'll appear), and the founder's availability window.
- Lock the **shot list** (Sections 6–7) and assign a single on-site "shot captain" to
  tick each frame off.
- Arrange model-release forms (Section 11) and brand assets (logo decals, branded
  linen bags, uniforms, signage).

**2–3 days before**
- Deep-clean + declutter the unit (Section 4). Order/clean uniforms. Charge batteries,
  format cards, test lights.
- Prep "hero linen": a set of **pristine, brand-white** towels/sheets reserved only for
  the shoot (never client linen — see §11).

## 4. Location prep — make the unit photogenic

A laundry floor reads as "industrial" — we want **clean, bright, organised, premium**.

- **Deep clean** floors, machine fronts, glass doors, walls. Wipe lint, detergent
  residue, water marks. Shiny stainless = premium.
- **Declutter:** hide hoses, buckets, mops, random bins, personal items, sticky notes,
  cardboard. Coil/clip stray cables.
- **Brand it:** TLB logo decal on a clean wall or machine, branded laundry bags stacked,
  uniforms on staff, a small "ISO / process standards" board.
- **Stage linen:** neat folded stacks (faces aligned), a folding-in-progress pile, a
  trolley of crisp whites. Fold edges crisp.
- **Light:** clean/replace flickering tubes; aim for even, bright, slightly cool-white
  to match brand blue. Open any windows/blinds for soft daylight.
- **De-steam** the frame before stills (steam = soft, hazy photos) unless we *want* one
  intentional steamy hero.
- **Color pop:** a touch of brand blue in frame (a bag, a sign, a uniform accent) ties
  every photo to the palette.

```
  Floor staging map (bird's-eye):
  ┌──────────────────────────────────────────────┐
  │ [washers row] » hero dolly track along here    │
  │   ● operator loading                            │
  │                      [folding table]            │
  │   [dryers]            ● staff folding whites     │
  │                                   ▢ detail: hands+linen
  │ [press/finishing]    [trolley of crisp whites]  │
  │   ● ironing/steam                  → dispatch/bags
  └──────────────────────────────────────────────┘
```

## 5. Equipment list

**Photo**
- Full-frame camera (e.g. Sony A7 IV / Canon R6 / Nikon Z6) — 24MP+ for big crops.
- Lenses: **16–35mm** (wide floor/hero), **35mm & 50mm** (people/process),
  **90–100mm macro** (textures, dials, water droplets).
- Tripod, polariser (kills glass/stainless glare), lens cloths (steam!).

**Video**
- 4K-capable camera (same body fine), **gimbal** (DJI RS-class) for moving b-roll.
- Variable ND filter (constant shutter under tube lighting — avoids flicker).
- Lav + shotgun mic for the founder/staff interview; headphones.
- Optional: slider, drone (only if exterior/loading-bay shots are allowed).

**Lighting**
- 2× LED panels (bicolor) + 1 softbox/diffuser; reflector. Battery packs.

**Misc**
- Extra cards + 2× storage, power strips/extension, gaffer tape, clamps, step ladder,
  lint rollers, white cotton gloves (handling whites), spare clean uniforms.

## 6. The photo shot list

> Aim: ~25–35 keeper stills. Each scene = 1 wide master (16:9) + 1 tighter (4:3) +
> 1 detail (macro). Shoot brackets for the darkenable bands.

**A. Hero / establishing (16:9)**
- `▢` Wide of the machine row with an operator mid-task, depth + leading lines. `»` slight low angle.
- `▢` Bright full-floor overview showing scale and order.
- `▢` One intentional "steam + light beam" mood frame.

**B. Core services (4:3 — feed service cards)**
- `▢` **Washing:** operator loading a front-load washer, brand bag in frame.
- `▢` **Drying/finishing:** linen tumbling / pulling warm linen out.
- `▢` **Pressing/ironing:** flatwork ironer or hand-steam with crisp sheet.
- `▢` **Dry cleaning:** garment on the line / delicate handling.
- `▢` **Folding:** two hands folding a white towel, aligned stack behind.
- `▢` **Dispatch:** branded bags/trolley staged for delivery.

**C. How-it-works (4 matching 4:3, consistent framing)**
- `▢` Assess (clipboard/tablet, counting linen) · `▢` Process (machines) ·
  `▢` Finish (folding/press) · `▢` Deliver (bags/van/trolley).

**D. Sector proof**
- `▢` **Healthcare:** pristine whites, gloved handling, "infection-safe" feel.
- `▢` **Hospitality:** five-star folded sheet stack, banquet napkin fan, towel art.

**E. Equipment & standards**
- `▢` Hero of the machine bank (16:9). · `▢` Macro: dials/controls, the ISO/standards board.
- `▢` Macro textures: water droplet on fabric, clean white weave, neat fold edge.

**F. People & founder**
- `▢` Founder Shourya Jain on the floor (4:3 + 1:1 portrait), approachable, in uniform/smart.
- `▢` Candid team-at-work (authentic, not stiff). · `▢` A genuine smile/handover moment.

**G. Utility**
- `▢` 2–3 low-detail wides with clean negative space (center/sides) for **text overlays**
  (counters, CTA bands, testimonials bg).

## 7. The video shot list + storyboard

**Deliverables:** one **45–60s hero film** (16:9), 3–4 **15–20s vertical reels** (9:16),
plus a **silent 8–10s hero loop** for the homepage background.

**B-roll to capture (5–10s each, on gimbal, shoot 2× the moves you think you need):**
- `🎥 »` push-in down the machine row · `🎥 »` reveal over a folding table ·
  `🎥` close on loading a washer · `🎥` warm linen pulled from dryer ·
  `🎥` steam rising off a press · `🎥` hands folding a crisp white ·

## 8. Lighting & camera specifications

**Photo**
- Format **RAW**, sRGB for web export. ISO as low as light allows (100–800).
- Whites: nail **white balance** (custom/grey card) so towels read clean, not yellow/blue.
- Aperture f/4–f/8 for sharp machines; f/2–2.8 for people/detail bokeh.
- Slightly **cool** grade to align with brand blue, but keep whites neutral.
- Shoot **brackets** (±1, ±2 EV) for frames that will be darkened for text.

**Video**
- **4K (3840×2160), 24 or 25fps**; 50/60fps for slow-mo b-roll (folding, droplets).
- Shutter ≈ 1/50 (180° rule); use **ND** to hold it → no fluorescent flicker/banding.
- Flat/Log profile if comfortable (more grade latitude), else neutral.
- Lock white balance per setup; record clean audio separately for the interview.
- Gimbal for all movement; tripod for interview + locked detail shots.

## 9. Styling, wardrobe & people

- **Uniforms:** clean, pressed, consistent; brand colours/logo visible. Closed shoes.
- **PPE where real:** gloves/aprons in healthcare-linen shots = authenticity + hygiene story.
- **Grooming:** tidy hair, no loud logos/jewellery, name tags straight.
- **Linen:** only pristine brand-white "hero" linen on camera; crisp folds, aligned edges.
- **Direction:** ask staff to *do their real task slowly*; capture genuine, unforced action.
  Avoid staged thumbs-ups. A few warm, natural smiles for people frames.
- **Diversity & dignity:** show the team respectfully; everyone consents (forms, §11).

## 11. Legal, consent & client confidentiality

- **Model releases:** signed by every identifiable person (staff, founder). Keep on file.
- **Client confidentiality:** **do not** show client-identifying linen, monograms, room
  numbers, branded hotel/hospital linen, or paperwork with client names. Use TLB hero
  linen. (B2B clients value discretion — this protects relationships.)
- **Location/brand permissions:** if the unit shares a site with a client, get written OK.
- **Drone:** only with permission + legal airspace; avoid if near hospitals/airports.
- **Licensing:** ensure the photographer/videographer assigns full usage rights to TLB
  (web, social, print, ads, perpetual) in the contract.

## 12. Post-production & delivery spec

**Photo**
- Cull → retouch (dust, lint, glare, stray cables missed on set). Keep it real — no AI
  face/scene fabrication. Consistent cool-clean grade; neutral whites.
- Export **master** JPEGs at full res (≥2400px long edge), sRGB.
- Then run them through the **existing pipeline** so the site gets WebP+JPEG `srcset`:
  drop masters in `media/` → extend `scripts/process-press-media.mjs` jobs → `npm run media:press`.
  Target widths already used by the site: **1600/1024/640** (heroes), **1200/800/400** (cards).

**Video**
- Edit hero film + reels; color grade to brand palette; add captions (accessibility +
  silent autoplay). Licensed/royalty-free music only.
- Export web masters, then transcode like the MoU clip:
  - `mp4` (H.264, `-movflags +faststart`) **and** `webm` (VP9/Opus).
  - Poster frame → optimised JPEG.
  - Homepage loop: short, **muted, no captions**, ≤ ~3–4 MB, `preload="none"` friendly.
- Keep originals as source-of-truth in the repo's media area.
